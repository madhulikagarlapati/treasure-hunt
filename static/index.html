Complexity

BFS: explores level-by-level. Complexity: O(V + E)

DFS: explores deeply before backtracking. Complexity: O(V + E)

A*: uses a heuristic for best-first search. Complexity: O(E) in the best case, and O(bd) in the worst case depending on heuristic quality.
